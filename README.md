# Smart-Sales-Insights-Hub-Using-SQL-Power-BI
Smart Sales Insights Hub is a complete sales analytics solution that extracts, cleans, analyzes, and visualizes sales data using SQL and Power BI. The project connects to a structured sales dataset, performs data cleaning and exploratory analysis with SQL queries, and builds interactive dashboards in Power BI to actionable business insights.
